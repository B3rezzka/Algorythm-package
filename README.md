# Algorythm-package, made for lab #3 MAI
## Введение

**ALGORITHM-PACKAGE** — это мини-пакет алгоритмов и структур данных, реализованный на Python. Он предоставляет базовые функции, алгоритмы сортировки, структуры данных и вспомогательные инструменты, предназначенные для изучения и применения основных концепций алгоритмов.

## Возможности

- **Функции факториала и Фибоначчи:**
  - `factorial(n: int) -> int`: Вычисляет факториал числа `n` итеративно.
  - `factorial_recursive(n: int) -> int`: Вычисляет факториал числа `n` рекурсивно.
  - `fibo(n: int) -> int`: Вычисляет `n`-ое число Фибоначчи итеративно.
  - `fibo_recursive(n: int) -> int`: Вычисляет `n`-ое число Фибоначчи рекурсивно.

- **Алгоритмы сортировки:**
  - `bubble_sort(a: list[int]) -> list[int]`: Сортировка пузырьком.
  - `quick_sort(a: list[int]) -> list[int]`: Быстрая сортировка.
  - `counting_sort(a: list[int]) -> list[int]`: Сортировка подсчетом.
  - `radix_sort(a: list[int], base: int = 10) -> list[int]`: Поразрядная сортировка.
  - `bucket_sort(a: list[float], buckets: int | None = None) -> list[float]`: Сортировка корзинами (для чисел в [0, 1)).
  - `heap_sort(a: list[int]) -> list[int]`: Сортировка кучей.

- **Структуры данных:**
  - `class Stack`: Стек с операциями `push`, `pop`, `peek`, `is_empty`, `__len__`, и `min` (за O(1)).
  - `class Queue`: Очередь с операциями `enqueue`, `dequeue`, `front`, `is_empty`, `__len__`.

- **Вспомогательные функции (генерация тест-кейсов):**
  - `rand_int_array(...)`: Генерирует случайный массив целых чисел.
  - `nearly_sorted(...)`: Генерирует почти отсортированный массив.
  - `many_duplicates(...)`: Генерирует массив с большим количеством дубликатов.
  - `reverse_sorted(...)`: Генерирует массив, отсортированный в обратном порядке.
  - `rand_float_array(...)`: Генерирует случайный массив вещественных чисел в [0, 1).

## Допущения и ограничения

- Реализации сортировок не используют встроенные `list.sort()` или `sorted()` (внутрення сортировка в  `bucket_sort` использует самописную сортировку).
- `bucket_sort` по умолчанию предназначен для сортировки вещественных чисел в интервале [0, 1).
- Структуры данных выбрасывают исключения (`ValueError`, `IndexError`) при попытке выполнить некорректные операции (например, `pop` из пустого стека).
- Проект разработан и протестирован с использованием `pytest`.
